Intro

Hypertext Markup Language (HTML) was created to meaningfully structure web documents and allow them to easily link to one another in an arbitrary fashion (as opposed to the more hierarchical formats of the time).

Let's take a look at how it works!

Why HTML?

(Review books for fundamental main points)

Modern HTML is focused on semantics, which is about giving meaning to the contents of the page. This is how browsers and assistive devices such as screen readers can parse a page to give users a Table of Contents or differentiate between the main content and navigation.

In conjunction with CSS and JavaScript it also provides context for when to apply certain styles or which elements to manipulate on the page.
